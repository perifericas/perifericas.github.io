surpresa!

Vi que que o arquivo é um zip, então descompactei, mas veio outro zip, e dentro dele, veio mais um zip e foi assim que percebi que poderia ser um trabalho manual muito grande pois não sabia até onde iria a brincadeira com os zips :D

Hora do Bash

Nessa hora fez sentido o nome do challenge então fiz esse script para resolver essa boneca russa do primeiro dia.

while true; do find . -name "*.zip" | while read filename; do unzip -o -d "`basename -s .zip "$filename"`" "$filename" && cd ./zip; done; if [ -f zip/flag.bin ]; then break; fi; done;

Perceba que o script para quando ele acha o flag.bin, antes de fazer esse script com break ele fazia ao infinito e além, no lixo que aparecia no terminal percebi que existia esse arquivo, então coloquei o break :D

Done! Have fun!
